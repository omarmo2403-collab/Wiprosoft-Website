# Wiprosoft Website

Modern website for Wiprosoft — a UK-based IT & Web Marketing Agency. Built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
src/
├── app/          # Pages and routes
├── components/   # UI components, layout, page sections
├── data/         # Static data (services, portfolio, blog, etc.)
├── lib/          # Utilities and animation variants
```

## Pages

- `/` — Home (hero, services, portfolio, testimonials, FAQ, blog, CTA)
- `/about` — Company story, values, team
- `/services` — Web development, digital marketing, branding & design
- `/portfolio` — Filterable project showcase
- `/blog` — Articles on web dev, SEO, and digital marketing
- `/contact` — Contact form and office info

## Build

```bash
npm run build
npm start
```

## Deployment

Deploy to Vercel, Netlify, or any Node.js hosting platform.
