# Design Analysis

Design patterns inspired by designproject.io for the Wiprosoft website rebuild.

## Core Design Direction

### Dark Theme
- Deep, rich dark backgrounds creating a premium, modern aesthetic
- Layered dark surfaces to establish visual hierarchy
- Light text on dark backgrounds for high contrast readability

### Bold Typography
- Large, impactful headings that command attention
- Strong typographic hierarchy to guide the user's eye
- Mix of weights for visual contrast (bold headings, regular body text)

### Scroll Animations
- Elements animate into view on scroll for an engaging experience
- Smooth fade-in and slide-up transitions
- Staggered animations for grouped elements (cards, lists)
- Parallax effects on hero sections and feature images
- Progress-based animations tied to scroll position

### Card-Based Layouts
- Content organised in clean, contained card components
- Consistent padding and spacing within cards
- Subtle hover effects (scale, shadow, border glow)
- Grid-based arrangements with responsive breakpoints

### Accent Color
- Primary accent: **#5955FF** (vibrant indigo-blue)
- Used sparingly for CTAs, links, highlights, and interactive elements
- Creates strong focal points against the dark background
- Gradient variations for additional visual interest

## Interaction Patterns
- Smooth page transitions
- Hover states with subtle animations
- Magnetic cursor effects on key interactive elements
- Loading states with branded animations
- Micro-interactions on buttons and form elements

## Layout Principles
- Generous whitespace (dark space) between sections
- Full-width hero sections with contained content
- Asymmetric layouts for visual interest
- Consistent grid system throughout
